비트와이즈 연산 : 비트를 기준으로 논리연산 하는것.

비트와이즈 and : & 연산자
비트와이즈 or : | 연산자 활용
왼쪽 쉬프트 : << 연산자 활용
오른쪽 쉬프트 : >> 연산자 활용


int v1 = 1;
int v2 = 3;

int bitwise_or = v1 | v2;

~result(bitwise_or) = 3
(int 형을 4비트로 가정 int = 32bit)
1 = [ 0, 0, 0, 1 ], 
2 = [ 0, 0, 1, 1 ]

(or = 둘 중 하나라도 true 라면 true반환)
각 비트마다 or 연산을 하여 [ 0, 0, 1, 1] 의 값이 된다.

int bitwise_and = v1 & v2;
